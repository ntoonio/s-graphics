Sosse-grafik

Hejsan! Det här är en liten tjänst för att enkelt skapa grafik i samma stil som partiets.

Det är ingen större tjänst än ett par knappar, så förvänta dig inte något snyggt gränssnitt :) Det kan också vara så att det inte funkar med alla webbläsare. Sidan är testad med Firefox, Safari, och Chrome.

Hur man gör

Typsnitt

I de flesta webbläsare behöver du installera typsnittet som används manuellt (förutom i safari vad jag vet). Du kan ladda ner typsnitten från socialdemokraternas sida.

Klicka på länkarna och ladda ner filerna. Installera dem sedan genom att bara klicka på dom. Om det inte funkar kan du googla på hur man installerar respektive format för ditt operativsystem. Du kan också mejla mig, min epost står längst ner.

Mac (TTF-format)

  • Kapra Neue Custom

  • Avenir LT Std Medium

  • Avenir LT Std Black

Windows (WOFF-format)

  • Kapra Neue Custom

  • Avenir LT Std Medium

  • Avenir LT Std Black

Beskrivning

  1. Klicka på ett av formaten ovan

  2. Fyll i alternativen

  3. Klicka "Generera"

  4. Klicka "Ladda ner"

Olika stilar

För att skriva i andra stilar än standardstilen, börja raden med ett utropstecken och sedan det nummer på stil du vill ha, direkt följt av text
